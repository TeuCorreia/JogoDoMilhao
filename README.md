# 🎮 Jogo do Milhão

Projeto de um jogo de perguntas e respostas inspirado no programa **Show do Milhão**, desenvolvido com HTML, CSS e JavaScript.

---

## 📌 Sobre o projeto

Este projeto consiste em um jogo interativo onde o jogador deve responder perguntas de múltipla escolha.

* A cada resposta correta, o jogador recebe um valor/pontuação 💰
* Caso erre, perde uma certa quantia da pontuação ❌

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🧠 Funcionalidades

* Sistema de perguntas e respostas
* Verificação de respostas corretas
* Sistema de pontuação
* Redirecionamento ao vencer o jogo
